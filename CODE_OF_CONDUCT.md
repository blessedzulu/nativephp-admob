# Code of Conduct

This project adopts the **Contributor Covenant v2.1** as its code of conduct.

The full text lives at the canonical source and is incorporated by reference:

- English: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
- Translations: https://www.contributor-covenant.org/translations/

By participating in this project (issues, pull requests, discussions, any community space tied to this repository) you agree to abide by it.

## Reporting

If you observe or experience behaviour that violates the code of conduct, please report it privately by emailing the maintainers at the address listed in the [project's GitHub profile](https://github.com/blessedzulu/nativephp-admob).

Reports are handled confidentially. We aim to acknowledge within 72 hours.

## Scope

The code applies in all project spaces and when an individual is representing the project in public.

## Enforcement

Maintainers may, at their discretion, warn, temporarily ban, or permanently ban a contributor whose conduct violates this code, following the **Enforcement Guidelines** section of the Contributor Covenant.
